Date Picker Component - Clear Button in Calendar Template

Enhanced Date Picker with Clear Button in Calendar
The clear button is now integrated into the calendar popup instead of the input group. When you open the calendar, you'll see a "Clear" button at the top of the calendar.

Usage Example:

<date-picker 
    label="Select Date" 
    [(ngModel)]="selectedDate" 
    [allowClear]="true"
    placeholder="Choose a date">
</date-picker>

Features:

  • ✅ Clear button appears in the calendar popup header
  • ✅ Only shows when allowClear=true and a date is selected
  • ✅ Automatically closes calendar after clearing
  • ✅ Styled with Bootstrap outline button
  • ✅ Hover effect with red color for better UX

Implementation Details:

  • Template: Uses NgBootstrap's contentTemplate feature
  • Styling: Custom CSS for calendar header and clear button
  • Functionality: Calls clearDate() method and closes calendar
  • UX: Clear button positioned at top of calendar for easy access
Code Changes Made:

HTML Template Updates:

  1. Removed clear button from input-group
  2. Added [contentTemplate]="customContent" to ngbDatepicker
  3. Created ng-template with calendar header and clear button
  4. Added conditional logic for showing clear button

CSS Updates:

  1. Added .custom-calendar styles
  2. Added .calendar-header with border and background
  3. Added .btn-clear-calendar with hover effects
  4. Styled clear button with red hover state
